## How to write a post

1. Copy this file inside `src/content/blog/` and rename it — the filename becomes the URL
   (e.g. `zno-fuel-ratio.md` → `/blog/zno-fuel-ratio/`).
2. Fill in the frontmatter above. `category` must be one of:
   *Nanotechnology, ZnO Nanoparticles, Materials Characterization, Scientific Data Analysis,
   AI in Research, NanoSami Development.*
3. Write your article here in Markdown: headings with `##`, **bold**, *italic*,
   lists, tables, code blocks, and images (put files in `public/images/blog/`).
4. Set `draft: false` to publish. While `draft: true`, the post is hidden.
