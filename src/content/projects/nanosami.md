NanoSami is my flagship project: scientific software for **deterministic, offline analysis** of nanomaterials and experimental data. The same input always yields the same output, and everything runs locally — no accounts, no uploads, no cloud dependencies.

It is designed specifically for laboratory environments where reproducibility, data sovereignty, and reliability on restricted machines matter most.

See the dedicated project page for full details, features, installation, and roadmap: [NanoSami](/nanosami/).
