## How to add a project

1. Copy this file inside `src/content/projects/` and rename it (the filename becomes the URL).
2. Edit the frontmatter above: title, description, technologies, links, status.
3. Put screenshots in `public/images/projects/` and reference them with
   `image:` (card thumbnail) and `gallery:` (project page images).
4. Write the project details here in Markdown — this text becomes the project page body.
5. Delete this instructional text when you're done.
